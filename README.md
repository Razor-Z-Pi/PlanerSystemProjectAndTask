# PlanerSystemProjectAndTask
Интеграция систем управления проектами и задачами
***

1.	Анализ типовых информационных систем: система управления проектами, система управления задачами.
2.	Описание программных модулей: модуль управления проектами, модуль системы управления задачами, модуль интеграции между системой управления проектами и системой управления задачами.
3.	Модель бизнес-процесса: создание проекта в системе управления проектами, назначение задач в системе управления задачами, отслеживание выполнения задач и связь с проектом в системе управления проектами.
4.	Методы интеграции: использование API для передачи информации между системами, настройка синхронизации данных между системами.
5.	Итогом работы является: Интеграция системы управления проектами и системы управления задачами позволит повысить эффективность работы команды и уменьшить вероятность ошибок.

------------------------
###Инструкция
**Для запуска понадобиться любой серверный комплекс с (apache/nginx, php, mysql/mariadb)**
1) `git clone` с проектом в папку для сервера;
2) Используйте скрипт `db.sql` для создание бд;
3) В `config.php` поменяйте параметры для подключение к БД;
4) Запустите проект через `index.html`;
5) Проект успешно запущен!!! Приятного пользования:)
